Experiencia

También fue parte de un equipo que trabajó para llevar a cabo un concierto en el lago mayor de Chapultepec, donde tuvo la tarea de crear una aplicación móvil que envía las coordenadas geográficas del usuario a una base de datos para después ser utilizadas como parámetros de la síntesis de sonido.
